How to Set Up Microsoft Outlook

  1. Open Outlook → File → Add Account
  2. Choose “Manual setup” → “POP or IMAP”
  3. Fill in:
    • Email: demo@your-name.com
    • Account Type: IMAP
    • Incoming mail server: imap.your-host.com
    • Outgoing mail server: smtp.your-host.com
    • Password: your password
  4. Click “More Settings” → “Outgoing Server” → Check “My outgoing server requires authentication”
  5. Click Next → Finish
